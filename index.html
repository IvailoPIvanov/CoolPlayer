Evolink Player Embed Code

Use the following code examples to embed Evolink Player in your webpage.
1. JS Includes (place in the HEAD section)
2. Create an HTML element to contain your video (place in the BODY section) 
<div id="playerElement" style="width:640px; height:480px; padding:0"></div>
3. Embed the Evolink Player (place after the HTML code from #2) 
<script type="text/javascript">
EvolinkPlayer.create('playerElement',
    {
    "license":"PLAY1-6zAQQ-njxRC-pnnbx-Cmt7E-tcCjj",
    "title":"",
    "description":"",
    "sourceURL":"http%3A%2F%2F82.119.84.155%3A1935%2Ffls%2FbntW-ffnv.stream%2Fplaylist.m3u8",
    "autoPlay":false,
    "volume":"75",
    "mute":false,
    "loop":false,
    "audioOnly":false,
    "uiShowQuickRewind":false
    }
);
</script>
